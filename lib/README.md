# lib — PlatformIO user library directory

Drop custom libraries here.  Each sub-directory is treated as an
independent PlatformIO library (auto-discovered).

Examples of what belongs here:
- A closed-source peripheral driver you don't want in the main tree
- A third-party library not already bundled (ChibiOS 3.0.5, LispBM, etc.
  are already in the project root)
- A hardware-specific extension that overrides a default source

See: https://docs.platformio.org/en/latest/librarymanager/workspacelibs.html
